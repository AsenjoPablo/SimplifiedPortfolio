About this project

This is a quick project made by Pablo to test the use of JQuery.

As you can see, you can add products to a shopping cart. This shopping cart keeps track of everything that is added to it. It will add the value of the item to the total â‚¬ spent.

  • If the product doesn't exist in the shopping cart, it will be appended as a new div. The product's cost will be added to the total cost and it will be shown when clicking "Checkout".
  • If the product already exists, it won't be appended as a new item: it will increase the amount by one, and multiply the item's total cost by the amount of items.
  • Checkout button toggles an animation showing all added items and their cost, which is updated in real time.
  • When an item is removed, bootstrap finds its real location and deletes the item from the Checkout list.
  • The item's value is internally located and withdrawn from the total cost of the operation.
  • Every time an item is removed, it is totally reset (quantity and cost)

I hope you have fun using this little project!
